Project 2: Math Shape Learner
By Aiden
This project is a fun way to learn shapes and visualize them. This is a project for teachers teaching math with shape areas and perimeters, and for students who want a quick way without much work from them to have fun learning shapes while being able to see them visually. This project is great for more visual learners, since this project is centered around automated visuals.
Usage Instructions:
First, decide which mode you want to use! I suggest start with classic mode since it is a little less distracting (by clicking the classic mode button on the menu screen).
Next, answer questions to get better at math (The visuals will pop up whether you get the question right or not)
Finally, after a bit of classic mode, you can go to shape drop mode by re-running the program or pressing the play again button upon death.
The goal of shape drop mode is to make your stack of shapes overflow!!

AI usage:
I used AI to make the questions and question answer set, but that's it. Also, something funny I would like to share was that the AI made a lot of errors and I had to correct some of them.
